# MZSA Scraper

Этот скрипт предназначен для сбора данных и изображений с официального сайта МЗСА (mzsa.ru).

## Установка зависимостей

Для работы скрипта требуется Python 3 и следующие библиотеки:

```bash
pip install -r requirements.txt
```

## Использование

Запустите скрипт:

```bash
python scraper.py
```

Скрипт создаст папку `output` и сохранит туда данные о прицепах и изображения.

## Структура выходных данных

```
output/
  <Название прицепа>/
    images/          # Изображения прицепа
    options_images/  # Изображения опций
    product.json     # Основная информация
    options.json     # Опции
```

## Интеграция с проектом

После скачивания изображений, их можно перенести в `frontend/public/images` и обновить `db.json` или `trailers.ts` ссылками на локальные файлы.
